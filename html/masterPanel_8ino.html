Detailed Description

Sketch for the Compressed Air Rocket Master Panel.

This sketch monitors the switches and buttons in the Master Panel, writes data to the LCD, and communicates with the Remote Panel and Pad using Zigbee

See also
LaunchPadPacket.h

LCD Layout This section of documentation describes the complete layout of the LCD Screen by the Master Panel program. The LCD screen is 16 x 2 characters. 

                      Character Columns
                                111111
                      0123456789012345
      Line #1         aa S C R Ppp Ex
      Line #2         bb LA RA RL LL

aa = pressure #1 and bb = pressure #2. Both are 2 digit integers.
S = safety on. S is removed when safety is off
C = compressor on, C is removed when compressor is off
R = remote enabled, R is removed when remote is disabled.
Ppp is P12 (both pads enabled). P1_ (only pad1 enabled). P_2 (only pad2 enabled)
Ex = Error and x is an integer from 0 to 9 (erro code)
LA = Local Arm, LA removed when local arm is off or not authorized.
RA = Remote Arm, RA removed when remote arm off or not authorized.
RL = Remote Launch, RL only on when remote launch is authorized and button pressed.
LL = Local Launch, LL only on when local launch is authorized and button pressed.

#define _AnalogButtons_   0

Define the Analog Input # for the momentary push buttons.

Master controller has 3 momentary push button switches (SW1, SW2, and SW3). All three of these switches have one terminal wired to a single resistor (R0). Each switch has the other terminal wired to analog pin 0 and a seperate resistor. SW1 is wired to R1, SW2 wired to R2, and SW3 wired to R3. R1, R2, and R3 each form a voltage divider with R0 and the resistor values are chosen to produce 3 distinct voltages at analog pin 0.

#define _ArmLedBit_   2

Define the Digital Output # for the Arm Switch LED.

Master controller has an Arm Toggle switch with build-in LED. This output does not control the LED. Instead it supplies power to the LED and switch combo. LED will not turn on without power. LED will turn on when it has power and the toggle switch is on.

#define _ArmSwitchBit_   12

Define the Digital Input # for the Arm Switch.

This input monitors the Master controller Arm Toggle switch (DPST).

#define _LaunchLedBit_   3

Define the Digital Output # for the Launch Button LED.

Master controller has a Launch Button with build in LED. This pin turns the built in LED on and off. The button does not control the LED.

#define _PressureScale1_   1
